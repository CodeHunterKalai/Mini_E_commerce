# Mini E-commerce

A simple full-stack e-commerce application that allows users to browse products, add them to a cart, and simulate checkout. This project demonstrates a basic online shopping platform with frontend, backend, and database integration.

---

## Features

- Browse products by categories
- Add/remove products to/from the cart
- View cart with total price
- Checkout simulation
- User authentication (signup/login)
- Responsive UI for desktop and mobile

---

## Tech Stack

- **Frontend:** React.js, HTML, CSS, JavaScript  
- **Backend:** Spring Boot / Rest Api
- **Database:** MongoDB 
- **Version Control:** Git & GitHub
